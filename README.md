# Living Dex Tracker

A modern, dark-themed Pokémon Living Dex tracker built with React and Tailwind CSS. Featuring glassmorphism design and real-time PokeAPI syncing.

![Project Preview](./public/readme.jpg)

## Features
* **Dark Mode:** Deep purple "Aether" aesthetic.
* **Animations:** Floating Pokémon sprites and smooth transitions.
* **Auto-Save:** Progress is saved to your browser's local storage.
* **Mobile Friendly:** Fully responsive grid layout.
